Honest answers.

The questions ops people actually ask before they put a credit card down.

What counts as an "app"?
One .clear file deployed to one URL. Database, pages, agents, and endpoints inside that file all count as one app. Spinning up a separate file at a different URL is a second app.
What happens if I exceed my MAU limit mid-month?
Your app keeps running. We email you the day you cross the limit, and the next month auto-upgrades to the higher tier. We never cut off traffic in the middle of a billing cycle.
Can I export my code if I cancel?
Yes. Clear is your source code — the .clear file plus the compiled JavaScript output. Download both any time, on any plan including Free. Self-host with one Node command. We never lock your code.
How does the AI credit work?
Token-based, billed against the included credit pool. Every Claude or OpenAI call from your agents draws from it. Top up any time at cost — no markup. You can also bring your own Anthropic / OpenAI API key on Team and above.
Is there an annual plan?
Yes — pay annually and get 2 months free (about 17% off). Same on Team and Business. Toggle "Annual" above to see the price. Enterprise contracts are quoted per company.
Self-hosted option?
Enterprise only. The compiler runs in your VPC or on-prem; your apps run wherever you want. We support Docker, Kubernetes, and a static binary. Source-available license, audited.
How does this compare to Retool?
Retool charges per developer seat. Ten devs touching internal tools = $500/mo on the Team plan, more on Business. Clear charges per company, not per seat — $99/mo flat for five apps. Plus Retool needs JavaScript and SQL; Clear needs English.
How does this compare to Lovable?
Lovable ships apps to Lovable's domain. Custom domain support is bolted on through Workspace add-ons, and the AI burns credits trying to fix bugs in code you can't read. Clear ships to your own domain on Team, and the source is English you can read and edit yourself.
