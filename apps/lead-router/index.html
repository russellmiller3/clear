Routing rules

Sales ops / routing

How leads are routed

Each new lead lands with one owner based on company size — and two named rules guard intake quality.

Owner assignment by company size

When a lead is submitted, its size field decides who picks it up.

SMB leads → alice

Mid-market leads → bob

Enterprise leads → charlie

No size set → alice (default)

Named rules the prover verifies on every push

lead-must-have-name — every lead needs a name field; blanks reject with HTTP 403.

lead-must-have-email — every lead needs an email; blanks reject with HTTP 403.

Both rules carry their name in the rejection body so the audit log attributes refusals by policy, not by line number.
