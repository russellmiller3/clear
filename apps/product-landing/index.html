You Can Read Every Line

This is what your invoice app actually looks like under the hood:

when user calls PUT /api/invoices/:id/send:
  requires auth
  define invoice as: look up the Invoice with this id
  guard invoice is not nothing or "Invoice not found"
  guard invoice`s status is "draft" or "Only drafts can be sent"
  invoice`s status is "sent"
  save invoice to Invoices
  send back "sent" with success message

Eight lines. You can read every one. "Require login, find the invoice, check it is a draft, change the status, save it." No React hooks. No middleware chains. No mysteries.
